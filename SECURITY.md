# Security Policy | ATMOSPHERIC OS™

## Supported Versions

Because Atmospheric OS™ is engineered for strict regulatory compliance, only the current active calibration cycle version receives official security updates and patches. 

| Version | Supported | Calibration Target |
| ------- | :---: | ------------------ |
| v2026.1.x | :white_check_mark: | Active 2026 Corporate Reporting Cycle |
| < v2026.0 | :x: | Deprecated / Legacy |

---

## Zero-Footprint Security Architecture

Atmospheric OS™ acts as an enforcement and auditing engine, not a data repository. 
* **Zero Client Data Retention:** Our infrastructure retains zero facility profiles, zero emissions logs, and zero case file histories.
* **Client Isolation:** All operational data persists exclusively inside your self-managed, geography-locked data environment (Supabase / Sovereign Cloud Node). 
* **Sovereignty Boundary:** If you suspect an infrastructure-level exploit within our core container engine, do not upload operational client data or system logs to this repository.

---

## Reporting a Vulnerability

We take the security of our sovereign auditing infrastructure seriously. If you discover a potential vulnerability within the core system container architecture, please report it immediately via the direct, secure channel below:

 **Email Reports directly to:** `legal@earthi.ltd`

### What to Include in Your Report:
1. A clear description of the vulnerability and the potential impact vector.
2. Step-by-step instructions or proof-of-concept code to reproduce the issue safely.
3. The specific engine version layer affected (`DRIFT GUARD™`, `RHYTHM ENGINE™`, etc.).

### Our Commitment:
* **Acknowledgment:** We will acknowledge receipt of your report within **48 hours**.
* **Triage:** A formal status update or patch timeline will be provided within **7 business days**.
* **Coordination:** We request that you adhere to responsible disclosure practices and refrain from publishing details of the vulnerability publicly until a patch has been coordinated and deployed to active ecosystem environments.
